Carpeta de imágenes
===================

hero-illustration.svg
----------------------
Ilustración corporativa abstracta (skyline + anillos), creada específicamente
para este proyecto, sin texto incrustado y sin licencia de terceros. Se usa
como recurso decorativo de respaldo.

Reemplazo recomendado para producción
--------------------------------------
Para la versión final se recomienda sustituir los fondos e ilustraciones por
fotografía corporativa real de INTER-CON Servicios de Seguridad Privada
(personal uniformado, instalaciones, oficinas), en formato .jpg/.webp,
optimizada para web (idealmente < 300 KB por imagen). Actualiza las
referencias correspondientes en index.html / styles.css.

No se incluyó la fotografía original de la presentación en este proyecto
porque no se pudo confirmar su titularidad/licencia de uso; sustitúyela por
material fotográfico propio de la empresa.
