Carpeta de iconos
=================

La mayoría de los íconos del sitio usan Font Awesome 6 (Free) vía CDN
(https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.5.1/css/all.min.css),
referenciado en el <head> de index.html. Esto mantiene el proyecto ligero
y evita almacenar cientos de archivos de íconos.

Esta carpeta contiene únicamente 2 íconos SVG propios, creados para este
proyecto (sin licencia externa):
- icon-shield-integrity.svg
- icon-handshake-trust.svg

Si deseas usar el logo/isotipo oficial de INTER-CON en formato vectorial,
coloca aquí el archivo .svg o .png provisto por la empresa y actualiza el
bloque ".brand-mark" en index.html para usar <img> en lugar del monograma
de texto "IC".
