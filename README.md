# Código de Ética - Landing Page Interactiva

Paquete listo para publicar en GitHub Pages.

## Uso rápido

1. Sube todos los archivos de esta carpeta a un repositorio de GitHub.
2. En GitHub ve a **Settings > Pages**.
3. En **Build and deployment**, selecciona **Deploy from a branch**.
4. Selecciona la rama `main` y la carpeta `/root`.
5. Guarda los cambios.

El archivo principal es `index.html`.

## Estructura

- `index.html`: contenido de la capacitación.
- `styles.css`: diseño corporativo.
- `script.js`: navegación, animaciones, acordeones, quiz y avance.
- `assets/slides/`: imágenes extraídas de la presentación base como recursos visuales.

## Edición

El contenido puede ajustarse directamente en `index.html`.
Los colores principales están definidos como variables CSS al inicio de `styles.css`.
